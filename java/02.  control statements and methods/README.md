# 제어문과 메서드

## 01. 제어문

- `조건문`: if문, switch문
- `반복문`: for문, while문, do~while문
- `분기문`: break문, continue문

## 02. 조건문

### 단순 if 문

> 💡 조건식이 `true일 때만` 실행문을 수행

```java
if(조건식) {
  실행문(들);
}
```

### if~else 문 

> 💡 조건식의 `true`나 `false`에 따라 다른 실행문을 수행

```java
if(조건식) {
  실행문(들)
} else {
  실행문(들)
}

```
